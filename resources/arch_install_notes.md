# Installing Arch Linux with full disk encryption

Make sure system is in UEFI mode (should get an output without errors) :
```
ls /sys/firmware/efi/efivars
```
## Connect to internet
```
iwctl
device list
station wlan0 scan
station wlan0 get-networks
station wlan0 connect <SSID>
```
